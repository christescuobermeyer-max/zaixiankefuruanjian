# 外卖在线客服 AI 桌面助手 —— 前端 UI 设计文档

> 版本：v1.0 · 日期：2026-06-28 · 风格：简洁专业 / 浅色
> 配套：`wireframes/`（SVG 线框图）、`项目开发文档.md`

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| 简洁专业 | 浅色界面、克制配色，适合客服长时间使用，降低视觉疲劳 |
| 信息密度适中 | 对话列表 + 聊天区双栏，关键信息（店铺名/时间/状态）一眼可见 |
| 即时反馈 | 流式打字、加载态、错误态都有明确视觉反馈 |
| 非侵入 | 默认隐藏于桌面顶部，不打扰；需要时一滑即达 |
| 一致性 | 统一的圆角、间距、字号阶梯、交互动效时长 |

---

## 2. 设计令牌（Design Tokens）

### 2.1 配色（浅色主题）

| 角色 | Token | 色值 | 用途 |
|------|-------|------|------|
| 主色 Primary | `--color-primary` | `#2563EB` | 主按钮、激活态、链接、品牌 |
| 主色浅 | `--color-primary-soft` | `#E3EBF5` | 选中对话项、用户气泡底 |
| 辅助/成功 | `--color-success` | `#16A34A` | 在线状态、成功提示 |
| 警告 | `--color-warning` | `#D97706` | 上下文将压缩提示 |
| 危险 | `--color-danger` | `#DC2626` | 登录错误、删除 |
| 背景 Base | `--bg-base` | `#EEF1F4` | 应用外层背景 |
| 表面 Surface | `--bg-surface` | `#FFFFFF` | 卡片、面板、气泡 |
| 表面次级 | `--bg-subtle` | `#F4F6F8` | 标题栏、侧栏 |
| 边框 | `--border` | `#D3D9DF` | 分隔线、描边 |
| 边框强 | `--border-strong` | `#9AA3AD` | 面板外框、聚焦输入 |
| 文本主 | `--text-primary` | `#2B313B` | 标题、正文 |
| 文本次 | `--text-secondary` | `#5A636E` | 标签、说明 |
| 文本弱 | `--text-muted` | `#8A929C` | 占位符、时间戳 |

> 助手气泡底 `#F1F3F5`，用户气泡底 `#E3EBF5`，区分发言方。

### 2.2 字体

- 字族：`"Segoe UI", "PingFang SC", "Microsoft YaHei", system-ui, sans-serif`
- 字号阶梯：`12 / 13 / 14 / 16 / 20 / 24`（px）
- 行高：正文 1.6，标题 1.3
- 字重：常规 400，强调/标题 600

| 用途 | 字号 | 字重 |
|------|------|------|
| 应用标题 | 20 | 600 |
| 区块标题（店铺名） | 14 | 600 |
| 正文/消息 | 14 | 400 |
| 标签/次要 | 13 | 400 |
| 时间戳/占位/提示 | 12 | 400 |

### 2.3 间距与圆角

- 间距基数：`4 / 8 / 12 / 16 / 24 / 32`（px，8 网格为主）
- 圆角：输入/按钮 `8px`，卡片/面板 `10–12px`，气泡 `10px`，标签 `6px`
- 阴影：面板 `0 8px 24px rgba(43,49,59,.12)`；弹窗 `0 12px 32px rgba(43,49,59,.18)`

### 2.4 动效

| 场景 | 时长 | 缓动 |
|------|------|------|
| 面板下滑/上滑 | 240ms | `ease-out` |
| 悬停揭示触发延时 | 200ms | — |
| 失焦自动隐藏延时 | 400ms | — |
| 气泡出现 / 按钮态 | 120–160ms | `ease` |
| 流式光标闪烁 | 1s 循环 | `steps` |

## 3. 布局规范

### 3.1 面板尺寸

- 宽度：**720px**，吸附屏幕顶部水平居中。
- 高度：默认 **500px**（可随内容/屏幕高度自适应，最大 80vh）。
- 无系统边框（`decorations:false`），自绘标题栏。

### 3.2 主面板栅格（参见 `02-main-panel.svg`）

```
┌──────────────────── 标题栏 44px（图标 / 标题 / 📌固定 / —隐藏） ───────────────────┐
├───────────────┬────────────────────────────────────────────────────────────────────┤
│ 侧栏 220px    │  聊天区 自适应（500-220）                                            │
│ ＋新建对话    │  对话头（店铺名 / 模型与压缩提示）                                   │
│ 对话列表       │  消息流（助手左 / 用户右 气泡，压缩分隔线）                          │
│ （店铺名+时间）│  ─────────────────────────────────────────                          │
│               │  输入框 + 发送                                                       │
│ 用户/退出      │                                                                     │
└───────────────┴────────────────────────────────────────────────────────────────────┘
```

## 4. 组件规范

### 4.1 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | 底 `--color-primary`，白字，圆角 8，hover 加深 8% |
| 次按钮（Ghost） | 白底，`--border` 描边，文本主色 |
| 图标按钮 | 28–32px 方/圆，hover 浅灰底 |
| 禁用态 | 透明度 0.5，`cursor:not-allowed`（如名称为空时「创建对话」） |

### 4.2 输入框

- 高度 40–44px，圆角 8，`--border` 描边；聚焦时描边 `--color-primary` + 2px 外发光。
- 占位符 `--text-muted`。
- 聊天输入：多行自增高，Enter 发送 / Shift+Enter 换行；发送中禁用并显示加载。

### 4.3 消息气泡（prompt-kit）

- 助手：左对齐，底 `#F1F3F5`，圆角 10（左上角小圆）。
- 用户：右对齐，底 `--color-primary-soft`，圆角 10（右上角小圆）。
- 流式：助手气泡内显示闪烁光标 `▍` + "正在输入…"。
- Markdown：支持分点、加粗、代码块（客服回复常用分点）。
- 上下文压缩分隔线：虚线 + 居中文案"— 较早对话已自动摘要压缩 —"。

### 4.4 对话列表项

- 高度 46px，圆角 7；激活态底 `--color-primary-soft` + 左色条。
- 两行：店铺名（14/600）+ 摘要/时间（12，`--text-muted`）。
- 悬停浅灰底；右键/悬停露出删除图标。

### 4.5 弹窗（新建对话，参见 `03-...svg`）

- 居中，宽 320，遮罩 `rgba(43,49,59,.10)`。
- 标题 + 单输入框（占位"建议直接填写店铺名，如「川味小厨」"）+ 取消/创建。
- 名称为空时「创建对话」禁用。

### 4.6 登录页（参见 `01-login.svg`）

- 居中卡片 360×400，应用图标 + 标题 + 邮箱/密码 + 登录按钮。
- 密码可显隐；错误态在按钮上方红字提示。
- 登录中按钮显示加载态并禁用重复提交。

## 5. 桌面交互视觉（Auto-hide / Edge Reveal）

参见 `04-hidden-hotzone.svg`。

| 状态 | 视觉 |
|------|------|
| HIDDEN | 面板移出屏幕上方，仅保留顶部居中 72×26 悬浮"客服"图标条（下圆角） |
| 悬停 | 图标 hover 200ms → 面板下滑显示；图标轻微高亮 |
| REVEALED | 完整面板，标题栏含 📌固定 / —隐藏 |
| PINNED | 📌 高亮，禁用失焦自动隐藏，便于持续客服 |
| 隐藏触发 | 失焦或鼠标离开面板 400ms → 上滑隐藏（240ms ease-out） |

- 下滑/上滑用 `transform: translateY()`，配合 Rust 窗口 `show()/hide()`。
- 顶部 2px 热区仅在面板水平范围内响应，避免全屏顶部误触。

## 6. 状态与反馈

| 状态 | 表现 |
|------|------|
| 空对话 | 聊天区居中提示"开启新对话，开始与客服 AI 协作" |
| 加载历史 | 骨架屏 3 条灰条 |
| 流式中 | 助手气泡光标闪烁 + 发送按钮禁用 |
| 网络/LLM 错误 | 气泡下方红字 + "重试"按钮（user 消息已存可重发） |
| 登录失败 | 表单红字提示 |
| 即将压缩 | 对话头黄色小标"上下文较长，将自动压缩" |

---

## 7. 无障碍与细节

- 颜色对比满足 WCAG AA（正文对背景 ≥ 4.5:1）。
- 所有可点击元素键盘可达（Tab/Enter）；输入框有 `label`/`aria-label`。
- 焦点可见（聚焦环），不仅靠颜色传达状态（错误同时给图标/文案）。
- 最小点击区 ≥ 32×32px。

---

## 8. 与实现的对应

- React 组件目录：`apps/desktop/src/features/{login,chat,panel}/`（每文件 ≤200 行）。
- Design tokens 建议落为 CSS 变量（`:root`）或 Tailwind theme，集中维护。
- prompt-kit 负责消息渲染与输入，Vercel AI SDK `useChat` 负责流式状态。
- 视觉风格、配色、间距以本文档令牌为准；线框图为布局/状态参照（非像素稿）。
