# 线框图索引

简洁专业浅色风格的低保真 SVG 线框图。直接用浏览器打开 `.svg` 查看。

| 文件 | 界面 | 要点 |
|------|------|------|
| `01-login.svg` | 登录界面 | 邮箱/密码 + Supabase Auth，错误态、密码显隐 |
| `02-main-panel.svg` | 主界面（顶部下拉面板） | 左侧对话列表（按店铺名）+ 右侧聊天区（流式气泡、压缩分隔线、输入框、固定/隐藏控件） |
| `03-new-conversation-modal.svg` | 新建对话弹窗 | 开启新对话前必填名称，建议填店铺名 |
| `04-hidden-hotzone.svg` | 隐藏态 + 顶部热区 | 自动隐藏、悬浮图标、悬停揭示、HIDDEN/REVEALED/PINNED 状态机 |

对应规范见 `../前端UI设计文档.md` 与 `../项目开发文档.md`（第 4 节面板交互）。
