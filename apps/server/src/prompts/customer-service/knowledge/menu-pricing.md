# 菜品 / 价格 / 分类 / 营业 / 产品结构

## 菜品/价格/分类/营业状态修改

"菜品和价格可以调，但不能看到问题就直接大改。价格动得太频繁平台和老客户都要重新适应，分类乱了顾客进店更难下单。您要加菜、改价、调规格或营业时间，可以把菜名、价格、分量、图片和想法发群里，我们结合店铺定位和转化数据处理。能直接改的会安排，涉及利润、活动叠加或平台规则的地方，我们先讲清楚利弊再动。"

## 半成品使用建议（成本控制）

"半成品对控制成本和提升出餐效率有帮助，关键是选对品类。汤底、酱料、预制菜这些质量稳定、出餐快，能在高峰期保持稳定供餐速度，这对店铺评分很重要。但要注意：口味要跟店铺定位匹配，不能为省成本影响体验；可以部分使用，基础汤底用半成品但配菜调味自己来；要算好账，有些半成品综合算下来不一定比自制划算。先选几个品类试，看顾客反馈和成本效益再决定扩大范围。"

## 产品特色 / 差异化优势分析

"您这个产品线规划得不错，差异化定位是优势，0添加0防腐这类卖点现在消费者很认可。引流款和复购款搭配得当，特价活动选成本可控的商品，做活动不亏本还能带动其他销量。后期上正餐类目能提升客单价、满足想吃正餐的客户，客户群体会更广。关于拍照，您能提供实物图最好，设计师可以基于真实产品做视觉优化，工艺特殊的产品实物图更能展现卖点。重点爆款的图片我们会优先处理，让顾客一眼看出您家产品的特色。"

## 盈利模式质疑 / 成本倒挂

"成功盈利的店铺有几个共同点：食材成本占比控制在30%以内、客单价合理不盲目打价格战、复购率保持40%以上。关键是算清真实成本结构（食材、人工、包装、平台扣点），倒推最低售价。现在平台更倾向扶持品质稳定、评分高的商家，而不是单纯低价店。建议重新定位，专注做好几道招牌菜提升客单价和利润率，而不是什么都卖什么都便宜。外卖如果做得对，利润空间比线下还好，因为租金成本低很多。"

## 自营优势 / 定价策略

"自营店在成本控制上有天然优势，没有加盟费、采购灵活、利润空间大。但平台不光看价格更看综合数据，单纯降价抢市场容易陷入价格战。平台更倾向推评分高、复购好的店铺。可以把成本优势用好但不是简单粗暴降价，比如特定时段做引流活动，或把招牌菜适当调低做引流，再通过套餐搭配和客单价提升保证整体利润。关键是让顾客感受到性价比优势，而不是让同行跟着一起降价。"

## 单量稳定后提升到手收入

"这个店数据跑得很稳，人群匹配和基础运营已经跑顺。接下来把节奏从要单量转到提到手：守住现在的曝光和转化，把每单利润抬上去。把高毛利招牌放首屏做规格升级和加价购，低毛利引流款权重降一点；价格和满减小幅细调，用进店小券拉点击、较高门槛满减保客单；远距离按区间抬起送门槛避免亏单；包装费和分量匹配到位。重点盯客单价、连带率、毛利率三项，趋势稳定后再微调，让到手持续往上走。"
