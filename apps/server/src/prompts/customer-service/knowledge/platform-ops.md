# 扣费 / 验证码 / 权重 / 拼好饭规则

## 扣费确认 / 收费规则说明

"咱们合作是按店铺实收订单金额的约定比例来扣，不是按原价、曝光或点击扣。订单里平台补贴、红包、包装费这些项目，后台展示口径有时看着比较绕，最终以服务市场和合同约定的结算口径为准。您对某笔扣费有疑问，可以把订单截图发群里，我们帮您对一下扣费来源和结算逻辑，避免您感觉钱扣得不明不白。"

## 验证码 / 后台授权 / 店铺掉线

"这个验证码是用于登录或绑定您当前合作店铺后台，方便我们继续处理店铺设置、活动、图片和数据这些运营工作，不涉及其他无关账号。后台掉线或平台要求重新验证时确实需要您配合。您确认是当前店铺的操作后，把验证码按约定方式发过来就行，我们登上去后继续把该处理的设置做完，处理完会在群里同步关键结果。"

## 店铺权重概念解释（通俗化）

"店铺权重就是平台给您店铺打的'信任分'，分越高平台越愿意把您的店推荐给顾客。平台每天给几千万用户推荐店铺，但首页位置有限，肯定优先展示表现好的店。权重高排名靠前、推荐时更容易上首页，曝光自然上去。平台判断权重主要看：销量、顾客评价、出餐准时度、有没有频繁退款投诉、店铺活跃度（参加活动、更新商品信息）。我们做代运营很多工作都是在帮您提升权重——优化商品信息、调整活动策略、提升好评率，让平台更信任您的店给更多流量机会。"

## 拼好饭上新品指导（平台规则）

"拼好饭跟主店规则不同，需按平台要求设置才能审核通过。核心要点：①价格上每个套餐至少保证2块利润，可按'成本+2块'定价，拼好饭虽是低价频道但不是让商家亏本做。②菜品优先选易批量制作、原材料成本低的，备餐效率高。③必须以套餐形式上架，至少3种，搭配一个引流款（如9.9爆款）加几个利润款。④拼好饭不支持红包、满减、优惠券，定价就是最终支付价，要把优惠提前考虑进去。⑤资料要齐全（高清图、详细描述、准确价格），审核约3天，资料不全会被退回。⑥商品分低于4.4分会失去爆品资格，分量足、口味好、包装整齐要做到位。拼好饭适合引流冲量，通过品质和服务把新客转化成主店回头客。"

## 拼好饭订单归属质疑（主店运营价值）

"拼好饭和主店共享一个店铺账号，权重、评分、商品数据都是相通的。我们优化主店的关键词、商品权重、价格、图片，直接影响整个店铺的排名和推荐，拼好饭频道也会受益。平台给拼好饭分配流量时会参考店铺整体数据表现——主店的评分、出餐速度、销量、评价。主店基础数据优化好、权重提升了，平台就会在拼好饭频道给更多曝光。拼好饭订单增长正说明主店运营有效果。这是相辅相成的关系，不是割裂的，我们会继续优化主店各项数据。"
